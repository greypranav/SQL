Create a user with password

CREATE USER 'user1'@'localhost' 
IDENTIFIED BY 'grey'; 
GRANT CREATE, ALTER, DROP
ON new_app.*
TO 'user1'@'localhost';
GRANT SELECT, INSERT, UPDATE, DELETE ON new_app.*
 TO 'user1'@'localhost';

  • This is a set of SQL commands used to create and configure permissions for a user in a MySQL or MariaDB database system.

#1) CREATE USER 'user1'@'localhost' IDENTIFIED BY 'grey';

  • This command creates a new database user named user1.
  • The localhost part specifies that this user can only connect to the database from the same machine where the database server is running (i.e., local connections only).
  • IDENTIFIED BY grey sets the password for this user to "grey".

#2) GRANT CREATE, ALTER, DROP ON new_app. TO 'user1'@'localhost';*

#3) GRANT SELECT, INSERT, UPDATE, DELETE ON new_app. TO 'user1'@'localhost';*
